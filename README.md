# Smart Property Price Prediction System

## Overview
This project predicts residential property prices using Machine Learning.

## Features
- Dataset Analysis
- Interactive Visualizations
- Model Comparison
- Housing Map
- Property Price Prediction
- Prediction History
- Market Insights

## Technologies
- Python
- Streamlit
- Pandas
- Scikit-Learn
- Plotly

## Run Locally

pip install -r requirements.txt

streamlit run app.py
